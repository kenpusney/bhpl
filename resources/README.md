# 📚 资源

资源的命名大致遵循以下的格式（除主标题外，其余皆有可能省略）：

```
<文档编号>: <主标题> - <副标题> (<版本>) - <作者>, <会议/文集>, <时间>
```

部分论文可能被收录在某些学术刊物出版社（如 Springer）合集中，不太容易找到免费资源。ACM Digital Library 等资源库的论文也有很大一部分需要付费获取。如果你有需要，可以私信我。
